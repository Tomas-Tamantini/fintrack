# FinTrack

FinTrack is a simple Python application to help you track your personal finances.  

## Getting Started

1. **Requirements:**  
   - Python 3.13 or higher

2. **Install dependencies and run with UV:**
   ```bash
   uv run task run 
   ```

3. **Run the tests:**
   ```bash
   uv run task test
   ```

4. **Run the linter:**
   ```bash
   uv run task format
   ```
